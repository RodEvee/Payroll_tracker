üêã METHOD 3: DEPLOY WITH DOCKER
================================

Step 1: Create Dockerfile
--------------------------
Create file named "Dockerfile" with this content:

FROM python:3.9-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt
COPY . .
EXPOSE 8501
CMD ["streamlit", "run", "app.py", "--server.address=0.0.0.0"]

Step 2: Build and run
---------------------
docker build -t payroll-tracker .
docker run -p 8501:8501 payroll-tracker

Step 3: Access
--------------
Open: http://localhost:8501

üí° TROUBLESHOOTING
==================

Problem: ModuleNotFoundError
Solution: pip install -r requirements.txt

Problem: Port 8501 already in use
Solution: streamlit run app.py --server.port 8502

Problem: Import errors
Solution: Make sure you're in project root directory

Problem: Changes not showing
Solution: Refresh browser or restart Streamlit

Problem: Data not persisting
Solution: Expected - uses session state. Add database for persistence.
